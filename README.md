# SuiLatam.xyz 🌎

**Tu Comunidad Latina en el Ecosistema Sui**

SuiLatam es la primera comunidad dedicada a conectar, educar y empoderar a los hispanohablantes y latinoamericanos en el ecosistema Sui blockchain. Rompemos las barreras del idioma y creamos contenido relevante para nuestra región.

![SuiLatam Banner](https://res.cloudinary.com/das1ozwma/image/upload/v1748645630/WhatsApp_Image_2025-05-30_at_19.28.29-removebg-preview_w20ozk.png)

## 🚀 Misión

Democratizar el acceso a las finanzas descentralizadas y la tecnología blockchain para la comunidad latina, proporcionando:

- **Educación** en español sobre Sui blockchain
- **Onboarding** simplificado para nuevos usuarios
- **Herramientas** financieras accesibles
- **Comunidad** activa y colaborativa
- **Recursos** culturalmente relevantes

## ✨ Características

### 🌐 Sin Barreras de Idioma
- Contenido 100% en español

- Desde principiantes hasta avanzados

- Webinars y workshops

### 🤝 Comunidad Activa
- +2,500 miembros en 15+ países
- Discord y Telegram activos
- Meetups presenciales

### 🛠️ Recursos Prácticos

- Herramientas de desarrollo

## 🚦 Instalación y Desarrollo

### Requisitos
- Navegador web moderno
- Editor de código (recomendado: VS Code)
- Git

### Clonar el Repositorio
```bash
git clone https://github.com/suilatam/suilatam.xyz.git
cd suilatam.xyz
```

### Desarrollo Local
```bash
# Abrir con Live Server (VS Code)
# O simplemente abrir index.html en el navegador
open index.html
```


1. Fork el repositorio
2. Crea una rama para tu feature (`git checkout -b feature/nueva-funcionalidad`)
3. Commit tus cambios (`git commit -m 'Agregar nueva funcionalidad'`)
